# BuddyBuddy
![image](https://github.com/extrayugen/BuddyBuddy/assets/59593892/0a45b49f-3293-4046-a7c4-1420bd62d34c)

## Introduction
- `BuddyBuddy`는 옛날 인기 있었던 '버디버디' 메신저를 모티브로 한 iOS 애플리케이션입니다.
- 이 앱은 추억의 메신저를 현대적인 모바일 경험으로 재해석하였습니다

## Stack
- SwiftUI
- Core Data
- MVVM 
- Gen-2

## Font
- Pretendard
- Jalnan2
